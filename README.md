# 🌍 World Project

## About

This repository is used to demonstrate the basic concepts of GitHub, VS Code, and Python.

We will build a simple "world" together by adding elements like houses, paths, and other structures.

---

## Goal of this session

* Understand how GitHub helps people work together
* Make a small change using a branch
* See how Python can interact with our world

---

## Project Structure

* `world_project/village.txt` → our world (simple text representation)

---

## Your First Task 🚀

1. Open `village.txt`
2. Create a new branch (e.g. `feature/add-house`)
3. Add something to the world (house, tree, shop, etc.)
4. Commit your change

---

## Python (later in the session)

We will use Python to read and display our world:

```bash
python show_world.py
```
